# pairmarket

Prediction markets on relationship outcomes. Friends invite each other to
private markets — "will A and B last 3 dates?" — settled on Sui, with
private content stored on Walrus and access mediated by SEAL. Sign in
with Twitter to play; a custodial wallet is created for you.

## Status

Pre-spec. Read [.planning/BRIEF.md](.planning/BRIEF.md) for the vision,
then [AGENTS.md](AGENTS.md) for the working rules.
